Privacy & Data Policy

Last updated: 2026-05-12

What LooFinder does

LooFinder helps people find public toilet facilities nearby. To improve quality and reliability, we store limited app data such as reviews, feedback, and aggregated usage analytics.

Data we collect

  • Reviews: Facility rating, review text, facility ID, and timestamp.
  • Feedback submissions: Message text, timestamp, and optional email if you choose to include it.
  • Analytics events: App interaction events (for example map load, filter toggles, directions clicks, submit success/failure), app version, and anonymous session ID.
We do not require account sign-up to use the app.

How we use data

  • Show review summaries and improve facility quality signals.
  • Respond to feedback and fix issues.
  • Monitor app reliability, performance, and feature usage through aggregated analytics.
  • Prevent abuse and moderate unsafe content.

Data sharing and storage

  • Data is stored on backend infrastructure used by LooFinder.
  • We do not sell personal data.
  • Data may be processed by service providers used to run the app (for example hosting, database, and email delivery).

Your choices

  • You can use LooFinder without submitting reviews or feedback.
  • Providing an email in feedback is optional.
  • You can request review/feedback removal for data you submitted.
Data requests / privacy questions Use the in-app feedback form and include the phrase Privacy Request in your message so it can be prioritized.

Security and content safety

  • Input is sanitized and filtered to reduce harmful content and script injection risk.
  • Rate limiting is used to reduce abuse.
  • No system can guarantee absolute security, but we work to minimize risk and respond to issues quickly.
